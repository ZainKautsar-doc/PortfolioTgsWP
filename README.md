## Deskripsi Proyek
Website ini merupakan proyek portfolio pribadi yang dibuat sebagai bagian dari tugas mata kuliah **Web Programming**. Tujuan utama dari proyek ini adalah untuk menampilkan profil singkat, keahlian, resume, serta halaman kontak dalam bentuk website statis yang responsif dan terstruktur dengan baik.

Website ini dikembangkan menggunakan teknologi dasar web seperti **HTML, CSS, dan JavaScript**, dengan fokus pada tampilan modern, struktur kode yang rapi, serta pengalaman pengguna yang sederhana dan mudah dipahami.

Selain untuk memenuhi kebutuhan tugas akademik, website ini juga dirancang sebagai media **personal branding** untuk menampilkan kemampuan dasar dalam pengembangan website.

---

## Fitur Utama
- Halaman Home sebagai pengenalan profil
- Halaman About untuk informasi singkat tentang diri
- Section Skills untuk menampilkan keahlian
- Halaman Resume sebagai contoh CV
- Form Contact sebagai contoh struktur form web
- Desain responsif untuk berbagai ukuran layar
